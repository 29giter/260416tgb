由于中共中央总书记习近平在当下被指控形成空前的个人独裁体制，借反贪腐运动大量清除异己，例如当年7月和10月时任外交部长秦刚遭解职以及时任国防部长李尚福上任半年多即被解职，加上习近平任内一直打压李克强所属的共青团派，例如在中共二十大后将当时年仅59岁的胡春华清除出中共中央政治局[86]。同时身为自由派和改革派的李克强不支持习近平终结改革开放的倒退政策且权力被架空[87][88][89]，所以有阴谋论观点认为其死因和习近平有关，或者是“被心脏病”死亡[90][91][92]。民进党立委王定宇认为李克强之死是习近平对团派的政治清洗，更将胡锦涛二十大离场事件与李克强骤逝联想在一起[93]。民运人士王丹在Facebook指出，李克强之死“正常死亡的可信度，小于非正常死亡的可信度”[94]。《日经新闻》前中国分社社长中泽克二称得到消息：“如果习近平因为意外事件不得不提前退休，那么李克强就会接替他的位置”；习对李始终没有放松警惕[95][96]。新华社前广东分社社长、退休记者顾万明在网上发公开信要求调查死亡真相，于2024年11月被上海法院以寻衅滋事罪判处1年徒刑，并被新华社剥夺退休金[97]。
